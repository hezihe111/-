# 新装备、功法、词条与战斗清单

本版本保留上一版已经完成的联机同步、装备实例、完整装备栏、功法携带、宗门大比准备房间、世界奇遇同步等内容，并继续补齐剩余玩法。

| 系统 | 本版状态 |
| --- | --- |
| 宗门劫掠/抢徒 | 改为选择 1 名弟子，对战目标宗门驻守弟子，战报显示回合、功法、伤害、回血、护盾 |

## 熟练度线性成长

功法效果倍率：

`1 + 品级 * 0.18 + 熟练度 * (0.006 + 品级 * 0.0015) + 悟性词条百分比`

例：蓝色功法品级 2，熟练度 50，无悟性词条时，倍率为 `1 + 0.36 + 50 * 0.009 = 1.81`。
