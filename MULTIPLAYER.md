# 联机启动说明

## 本机测试

1. 双击 `start-multiplayer.bat`，或在当前目录运行：

   ```bat
   node server.js
   ```

2. 浏览器打开：

   ```text
   http://127.0.0.1:8097/
   ```

3. 开局菜单选择“联机模式”，输入房间号，例如 `ZIHE01`，点击“连接”。

## 手机或朋友加入

1. 所有人连接同一个 WiFi。
2. 房主电脑保持 `start-multiplayer.bat` 窗口运行。
3. 其他设备打开房主电脑的局域网地址，例如：

   ```text
   http://192.168.0.105:8097/
   ```

4. 选择“联机模式”，输入同一个房间号，点击“连接”。

如果手机打不开，请检查 Windows 防火墙是否拦截 Node.js，允许它在“专用网络”通信。

## 当前联机范围

- 同房间玩家会显示在中心地图上。
- 玩家可提交回合，服务器等待全员完成后一起进入下一年。
- 禁地探索会向其他玩家显示当前层数。
- 资源点、AI 宗门、普通机缘、边境状态会广播大世界快照，减少地图分裂。
- 玩家间可以进行基础掠夺/抢徒战报。

这是第一版轻量房间服务器，适合局域网和小规模测试。正式公网部署时建议放到云服务器，并用反向代理提供 HTTPS/WSS。
