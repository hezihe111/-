# 新手手册

## 开局

1. 选择单机或联机模式。
2. 输入宗门名，选择宗门印记。
3. 在地图上选择山门所在地。灵气影响弟子成长，资源影响经济，风险影响冲突。
4. 前几年优先完成：收徒、占资源点、探索机缘、建设议事殿或观星楼。

## 弟子养成

弟子详情里新增“弟子养成”入口，集中管理以下系统：

| 系统 | 说明 |
| --- | --- |
| 道心 | 弟子生成时确定，影响年度修行、战斗倾向、心魔和个人事件 |
| 体质 | 低概率天生出现；也可用洗髓探体尝试觉醒，但成功率很低 |
| 专精 | 剑修、体修、丹修、器修、阵修、毒修、雷修、阴阳修八类路线 |
| 师徒 | 低境界弟子拜高境界弟子为师，每年获得额外成长 |
| 闭关 | 可偏向修为、功法熟练度或问心降心魔 |
| 本命法宝 | 不占装备栏，可长期升级并提供战力 |
| 突破试炼 | 正式渡劫前积累加成，提高下一次成功率 |
| 个人事件 | 每年可主动触发一次，获得属性、资源、专精或心魔变化 |

## 战力理解

战力由境界、基础属性、装备、词条、功法、熟练度、道心、体质、专精、本命法宝、称号等共同决定。境界差距会明显拉开战力，符合越阶难度较高的修仙设定。

## 资源建议

- 前期灵石不要全部拍卖，留一部分给建设和炼制。
- 丹材现在可通过机缘、资源点、拍卖、流浪商人和世界奇遇获取。
- 洗练灵砂、定纹玉扣用于装备词条系统，追求红词条时消耗较大。
- 本命法宝需要器材和洗练灵砂，建议优先给核心弟子打造。

## 联机建议

当前定位是私下几个玩家的本地/小房间对战体验，不按大型网游设计。联机时重点体验统一拍卖、会议投票、宗门大比准备结算、世界奇遇协作、巨兽 Boss 共享血量、玩家交易和 PvP 房间。

## 新手指引

开局会弹出是否开启新手指引。开启后，前几年会高亮当前已解锁的重要区域，例如目标、收徒、弟子详情、弟子养成、仓库、地图、工坊和下一年按钮。也可以随时跳过。
