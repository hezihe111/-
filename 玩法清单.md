# 玩法清单

本清单对应当前“指引奖励修复版”，用于快速查看已经实装的玩法范围。

## 开局与引导

| 系统 | 内容 |
| --- | --- |
| 单机模式 | AI 宗门自动发展、占点、结盟、袭扰和参赛 |
| 联机模式 | 同房间玩家共用年份节奏，提交回合后等待其他玩家 |
| 宗门命名 | 可输入宗门名并选择宗门印记 |
| 山门选址 | 地图选址有灵气、资源、风险差异 |
| 宗门路线 | 正道盟宗、霸道战宗、商会仙门、隐修道统、魔道独行 |
| 终局目标 | 飞升结局、仙盟盟主、天下第一宗、商业霸主、阵法天庭、魔道独尊 |
| 新手手册 | 地图右上角问号打开完整说明 |
| 高亮指引 | 立宗后可选择开启，前几年随玩法解锁逐步高亮提示 |

## 年份与行动点

| 系统 | 内容 |
| --- | --- |
| 年份推进 | 每年选择年度天命，然后分配行动点 |
| 行动点 | 建设、探索、争夺、炼制、参悟、边境等操作消耗行动点 |
| 十年成长 | 每十年基础行动点上限提高 |
| 联机等待 | 联机玩家提交回合后等待全员完成 |
| 后加入同步 | 后进房间玩家会接收世界快照和正在进行的房间事件 |

## 弟子养成

| 系统 | 内容 |
| --- | --- |
| 弟子属性 | 体魄、攻伐、守御、身法、资质、心性、气运 |
| 境界成长 | 弟子修行满后触发渡劫或境界晋升 |
| 战力显示 | 弟子详情显示战力，境界越高战力跨度越大 |
| 核心弟子 | 可标记核心弟子，便于培养和排序 |
| 长老系统 | 金丹以上弟子可晋为长老，提供长期加成 |
| 羁绊系统 | 弟子之间可缔结羁绊，影响战斗和事件反馈 |
| 心魔系统 | 心魔会影响修行和风险，可消耗参悟压制 |
| 功法配置 | 每名弟子最多携带 5 本功法并调整顺序 |
| 功法熟练 | 熟练度线性提高功法效果和战力 |

| 建筑 | 效果 |
| --- | --- |

| 系统 | 内容 |
| --- | --- |

## 战斗系统

## 本版新增：本地/私房联机体验

本版目标转向“私底下几个玩家间的本地对战体验”和纯单机体验，不按大型网游方向设计。

| 系统 | 本版状态 |
| --- | --- |
| 联机 PVP 房间 | 快捷掠夺/抢徒不再只挑战公共状态投影。发起方选 1 名弟子，对方弹出挑战请求并选择 1 名守擂弟子，确认后由发起方统一生成实时斗法，双方播放同一份战斗动画与结果。 |
| PVP 拒绝返还 | 对方拒绝、盟友冲突等情况下，发起方返还 1 点行动点。 |
| 世界奇遇锁步 | 协作奇遇进入准备房后，由队长统一推进选项；队员看到同一组选项、同一步进度，不能各走各的，但仍可使用护持降低危险。 |
| 巨兽 Boss 房间 | 巨兽挑战升级为协作房。所有在线玩家各选最多 2 名弟子准备，房主在全员准备后统一开战，所有参与者播放同一场队伍战斗，伤害统一扣除共享血量。 |
| Boss 难度伸缩 | 巨兽同场战斗会随参战弟子数量提高血量/强度系数，避免多人直接碾压。 |
| 三档存档 | 保存/读取改为 1/2/3 三个栏位，旧单档存档兼容显示在 1 号档。读档会清理进行中的临时联机房间状态。 |
| 内容扩容 | 新增 29 本功法、21 件装备，进入拍卖、奇遇、商人、交易和奖励池。 |

## 本版新增：弟子养成玩法

| 系统 | 内容 |
| --- | --- |
| 道心 | 弟子生成时获得长期性格路线，影响年度修行、战斗倾向、心魔风险和个人事件收益 |
| 特殊体质 | 新弟子低概率天生拥有特殊体质；未显体质可通过洗髓探体尝试觉醒，但成功率保持较低 |
| 弟子专精 | 剑修、体修、丹修、器修、阵修、毒修、雷修、阴阳修八类路线，可在弟子养成弹框中改修 |
| 专精熟练度 | 年度修行、闭关、突破试炼、个人事件等都会增加熟练度，按 100 点一级线性成长 |
| 师徒传承 | 低境界弟子可拜高境界弟子为师，每年获得额外修为和专精经验；不启用羁绊组合 |
| 闭关修行 | 可安排 1/3/5 年闭关，分别偏向修为、功法熟练度或问心降心魔 |
| 本命法宝 | 消耗器材、灵石和洗练材料打造，不占装备栏，可长期升级并提供战力 |
| 突破试炼 | 正式渡劫前可做问心、引雷、杀伐、红尘试炼，成功后提高下一次渡劫加成 |
| 弟子个人事件 | 每名弟子每年可主动触发一次，也可能年度自然发生，奖励属性、专精、资源或心魔变化 |
| 称号外号 | 达成攻击、属性伤害、防御、边境战力等条件后自动获得称号，提供永久属性和战力 |
| 新手指引 | 手册新增弟子养成页，前几年高亮提示“弟子养成”入口 |
