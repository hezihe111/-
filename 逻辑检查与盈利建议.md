# 逻辑检查、补充玩法与盈利建议

## 本次检查到并已修复

| 问题 | 风险 | 处理 |
| --- | --- | --- |
| 巨兽奖励引用禁地遗物 `starShard` | `starShard` 不是普通仓库物品，作为巨兽奖励会入库失败 | 已替换为可入库的高阶功法、装备或洗练材料 |
| 玩家交易装备不保留词条 | 交易装备时只传装备 ID 和品质，接收方会得到随机新词条装备 | 已改为保留装备 UID、词条、锁定词条状态 |
| 旧物品可能错分仓库类型 | 旧存档或旧奖励 ID 可能让仓库分类混乱 | 已加入“待鉴定”兜底，不再误判成炼丹材料 |
| 开始界面信息层级弱 | 开局只像普通表单，不像游戏启动页 | 已改成左右布局：左侧玩法亮点和流程，右侧开局表单 |

## 当前仍需注意的逻辑边界

| 位置 | 当前状态 | 建议 |
| --- | --- | --- |
| 联机战斗 | 快捷掠夺仍是进攻方挑战对方公共状态投影，对方收到战报 | 若要真正 PvP，需要后端新增战斗房间、双方选人、双方确认结算 |
| 世界奇遇协作 | 可准备房、看队友状态、互助降危险，但选择仍偏本地推进 | 若要完全同步，需要后端锁步奇遇回合 |
| 巨兽 Boss | 共享血量和累计伤害已实现，但不是所有玩家同屏观战 | 可升级为 Boss 房间，所有玩家准备后同场播放 |
| 房主权威 | 拍卖、仙盟、大比等由房主统一结算 | 若商业化，要加服务端事件版本号、防重复结算、防乱序 |
| 交易系统 | 装备交易已保真，但索取栏仍只能按品级索取，不能指定词条 | 后续可做“上架市场”，直接展示具体装备和词条 |
| 旧存档兼容 | 已有兜底，但旧版本装备或旧字段仍可能弱化体验 | 可提供“一键整理旧存档”按钮 |

## 可补充玩法建议

| 优先级 | 玩法 | 价值 |
| --- | --- | --- |
| 高 | 玩家宗门集市/寄售行 | 玩家上架具体装备、功法、丹药，其他玩家用灵石购买，解决交易沟通成本 |
| 高 | 联机 Boss 房间 | 多名玩家选择弟子准备，同屏展示伤害排行和奖励，强化联机感 |
| 高 | 宗门赛季榜 | 每局或每周统计战力、财富、声望、Boss 伤害、大比冠军，增强回访 |
| 高 | 新手任务链 | 比现在任务板更细：收徒、占点、炼丹、装备、功法、第一次战斗逐步引导 |
| 中 | 宗门职位 | 掌门、丹堂、器堂、外务、执法、镇守长老分工，提供长期策略 |
| 中 | 宗门科技树扩展 | 剑修、丹修、器修、阵修、商修、魔修路线形成互斥或半互斥选择 |
| 中 | 秘境词缀 | 奇遇或边境副本带词缀，例如火脉、毒瘴、雷暴、禁疗，提高配队意义 |
| 中 | 宗门成就 | 首次金丹、首次红装、首次通关、首次联机大比冠军，提供永久展示 |
| 中 | 装备套装 | 同主题装备凑 2/4 件触发套装效果，提升装备收集目标 |
| 中 | 功法流派组合 | 火毒、金雷、水木、阴阳等组合触发流派加成 |
| 低 | 宗门皮肤 | 山门外观、地图旗帜、弟子头像框，不影响数值，适合商业化 |
| 低 | 离线收益 | 仅给少量资源或邮件，不破坏联机公平 |

## 是否可以盈利

可以，但建议先按“网页游戏原型”做轻商业化，不要一开始做强付费数值。最稳的路线是广告、赞助、会员、外观付费、创作者支持。不要做现金抽奖、可提现道具、赌博式玩法或带真实货币回报的随机奖励。

## 推荐盈利方式

| 方式 | 推荐度 | 做法 |
| --- | --- | --- |
| 页面广告 | 高 | 使用 AdSense 等广告平台，在非战斗核心区域放少量广告，不遮挡 UI |
| 赞助/打赏 | 高 | 放爱发电、Ko-fi、Patreon、微信/支付宝赞助码等，不影响平衡 |
| 高级会员 | 高 | 云存档、更多存档位、房间装饰、统计面板、战报导出 |
| 外观付费 | 高 | 宗门旗帜、山门皮肤、称号、头像框、战报皮肤 |
| 资料片式买断 | 中 | 新地图、新剧情、新 Boss、新功法包，避免卖强数值 |
| 服务器月卡 | 中 | 为联机房间、排行榜、赛季记录收费，适合有稳定玩家后 |
| 付费道具 | 低 | 如果做，只建议外观或便利，不建议卖红装、战力和抽卡 |

## 不建议的盈利方式

| 方式 | 原因 |
| --- | --- |
| 现金抽奖、赌博、可提现玩法 | 支付和广告平台通常限制，高合规风险 |
| 氪金直接买战力 | 容易破坏联机公平，玩家口碑差 |
| 强制看广告推进回合 | 影响游戏节奏，容易流失 |
| 未成年人诱导充值 | 高风险，需要防沉迷、退款、监护等更多合规设计 |

## 合规注意

| 场景 | 注意 |
| --- | --- |
| 使用广告 | 需要原创内容、可导航页面、良好用户体验，避免诱导点击 |
| 使用收款 | 支付服务通常限制赌博、抽奖、可提现游戏和高风险虚拟物品 |
| 面向中国大陆并使用大陆服务器 | 通常需要 ICP 备案；经营性互联网信息服务还可能需要 ICP 许可证 |
| 用户账号/云存档 | 要准备隐私政策、用户协议、数据删除方式 |
| 联机聊天或玩家内容 | 需要举报、屏蔽、敏感内容处理 |
